üöÄ System Overview

This is a production-ready, high-performance embedding search system that combines four search methods in parallel: Tantivy (full-text), Semantic (ML embeddings), Symbol (tree-sitter AST), and BM25 (statistical). Built with Rust for performance and safety, featuring a 3-chunk context system that provides 55% better accuracy, and integrated with Claude Flow for SPARC methodology and agent coordination.
